ABOUT

  1. This website is created with the help of the following:
    • VS Code
    • Bootstrap Themes
    • Bootstrap CSS styles
    • Some core CSS
    • Javascript
  2. It is a single page website, in which a couple of posts are hardcoded, however the website can handle adding new posts by clicking on the "ADD NEW" button found on the navbar. The navbar has other two elements in it, the first being the Brand Name, which acts as a home button, and an about button, which leads to the page with details on how the website was created.
  3. Apart from posting blogs, the page also has a filter option, where keywords can be searched, and the matched posts are shown while the rest of the post are filtered. The filter function is set to last only 5 seconds, after which the default view is loaded.
  4. Also there is a dedicated spot for ads and promotions.
  5. Future plans:
    • Implementation of authentication of users, so that only admins can post and delete the posts.
    • Connection to database.
    • Implementation of tags, for easier sorting of posts.
    • Uploading, storing and displaying images along with posts.
